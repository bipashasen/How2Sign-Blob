# Video-VQVAE
My PyTorch implementation of https://arxiv.org/abs/2103.01950

Work in progress


Implementations of 3D versions of:
- PixelSNAIL (only for training) https://arxiv.org/pdf/1712.09763.pdf
- VQ-VAE-2 https://arxiv.org/abs/1906.00446
